# .github

Default [community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file) for **GetCircuitBent**. Anything here applies to every GetCircuitBent repository that does not ship its own copy.

## `FUNDING.yml`

Defines the account-wide **Sponsor** button: GitHub Sponsors and Ko-fi (accepts Apple Pay / Google Pay). Individual repos should **not** carry their own `FUNDING.yml` — they inherit this one. See `Funding and Sponsorship.md` in the GitHub-Style-Guide repo for the standard.
